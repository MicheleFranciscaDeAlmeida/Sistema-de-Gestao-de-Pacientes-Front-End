# 🏥 Sistema de Gestão de Pacientes

## 📌 Sobre o projeto
Este projeto foi desenvolvido com o objetivo de simular um sistema de cadastro e gerenciamento de pacientes, permitindo organizar informações de forma simples, clara e eficiente.

## 🎯 Problema que resolve
Em clínicas e consultórios, a organização inadequada de dados de pacientes pode gerar perda de informações, atrasos no atendimento e dificuldades no controle dos registros.  
Este sistema simula uma solução digital para esse problema, centralizando e facilitando o acesso às informações.

## 🎯 Objetivo
Praticar conceitos de desenvolvimento front-end, organização de dados e criação de interfaces voltadas para usabilidade.

## 🚀 Funcionalidades
- Cadastro de pacientes
- Exibição de informações
- Manipulação de dados
- Armazenamento com LocalStorage
- Interface organizada e intuitiva

## 🛠 Tecnologias utilizadas
- HTML
- CSS
- JavaScript

## 💡 Diferenciais
- Foco em experiência do usuário (UX)
- Interface limpa e responsiva
- Simulação de sistema aplicado ao mundo real
- Estrutura pensada para evolução futura (back-end/API)
