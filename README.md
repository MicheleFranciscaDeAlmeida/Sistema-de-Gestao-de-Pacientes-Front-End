# 🏥 Sistema de Gestão de Pacientes

Aplicação web desenvolvida com foco em simular um sistema real de cadastro e gerenciamento de pacientes, utilizando tecnologias de front-end.

---

## 🚀 Funcionalidades

* ✔ Cadastro de pacientes
* ✔ Edição de informações
* ✔ Exclusão de registros
* ✔ Listagem em tabela profissional
* ✔ Busca dinâmica por nome
* ✔ Armazenamento com LocalStorage
* ✔ Interface responsiva (mobile e desktop)
